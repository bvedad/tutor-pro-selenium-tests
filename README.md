# Selenium tests for Tutor Pro

University project from Software Verification, Validation and Testing on Burch University. 
Selenium tests for Tutor Pro web application.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software:  
  
Java (version 1.7+)  
Maven


## Running the tests

Open project in IntelliJ, import maven dependencies and run tests.


## Built With

* [Maven](https://maven.apache.org/) - Dependency Management
* [Selenium](https://www.seleniumhq.org/) - Framework for automated testing
* [JUnit](https://junit.org/) - Unit testing framework for Java
